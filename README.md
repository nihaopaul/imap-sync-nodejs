imap-sync-nodejs
================


0. install: nodejs and npm
1. npm install -g bower
2. npm install
3. cd public; bower install
4. nodemon bin/www
5. install modules for imapsync: imapsync/examples/install_modules_linux.sh
6. http://localhost:3000


Only tested on OSX and CentOS, YMMV
